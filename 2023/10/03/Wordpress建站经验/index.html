tree命令展示指定深度文件夹

安装

sudo apt install tree

使用

直接使用，会显示指定文件夹下的所有文件：

tree 路径

指定显示深度, 此处为3级：

tree -L 3 路径

只显示目录，不显示文件：

tree -d -L 3 路径
